## Rodar o projeto

Para rodar o projeto pasta utilzar o seguindo comando: 
```yarn && yarn start```

## Tecnologias utilizadas

As seguintes tecnologias foram utilizadas:
- HTML
- CSS 
- Javascript
- Backend: PHP com framework CodeIgniter
- Banco de dados: MySQL
- FrontEnd: ReactJS
